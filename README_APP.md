# Приложение "Auth & UI Kit"

Современное веб-приложение на React и TypeScript, демонстрирующее возможности кастомной библиотеки компонентов и системы авторизации.

## Основные функции
- **Система входа**: Полноценная форма авторизации с валидацией данных.
- **Профиль пользователя**: Отображение персональных данных после успешного входа.
- **Библиотека компонентов (UI Kit)**: Витрина всех разработанных элементов интерфейса в различных состояниях.
- **Адаптивный дизайн**: Приложение корректно работает на мобильных устройствах, планшетах и десктопах.
- **Плавные анимации**: Использование библиотеки Motion для улучшения пользовательского опыта.

## Структура проекта
- `src/components/` — библиотека базовых UI-компонентов.
- `src/pages/` — основные страницы приложения (`LoginPage`, `UIKit`).
- `src/App.tsx` — корневой компонент, управляющий логикой приложения.
- `src/index.css` — глобальные стили и конфигурация Tailwind CSS.

## Используемые компоненты
В проекте реализованы следующие элементы управления:
1. Кнопки с разными стилями и размерами.
2. Текстовые поля ввода с валидацией.
3. Выпадающие списки.
4. Чекбоксы и радио-кнопки.
5. Переключатели (Switches).
6. Поля выбора даты.
7. Многострочные текстовые поля.

## Технологический стек
- **Frontend**: React 19, TypeScript.
- **Styling**: Tailwind CSS.
- **Animations**: Motion (framer-motion).
- **Icons**: Lucide React (опционально).

## Как пользоваться
1. При первом запуске заполните форму на странице входа.
2. После нажатия кнопки "Войти" вы увидите свои данные в профиле.
3. Используйте кнопку "Открыть UI Kit" в верхнем правом углу, чтобы просмотреть все доступные компоненты.
4. Вы можете выйти из профиля, нажав кнопку "Выйти".
