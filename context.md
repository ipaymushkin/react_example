# Контекст проекта: Auth & UI Kit App

## Общая информация
Данное приложение является демонстрационной платформой, объединяющей систему авторизации и библиотеку пользовательских интерфейсов (UI Kit). Оно построено на современном стеке технологий с упором на производительность, типизацию и плавность интерфейса.

## Стек технологий
- **React 19**: Использование функциональных компонентов и современных хуков (`useState`, `useId`).
- **TypeScript**: Полное покрытие типов для компонентов, пропсов и данных пользователя.
- **Tailwind CSS**: Стилизация через утилитарные классы. Используется темная тема (`zinc-950`) с акцентами `emerald`.
- **Motion (Framer Motion)**: Реализация анимаций появления, переходов между страницами и интерактивных состояний.
- **Vite**: Сборка и быстрая разработка.

## Архитектура компонентов (`src/components/`)
Все компоненты являются атомарными и переиспользуемыми:
- **Button**: Поддерживает варианты (`primary`, `secondary`, `outline`, `ghost`), размеры (`sm`, `md`, `lg`) и состояние загрузки.
- **Input**: Текстовое поле с поддержкой лейблов и отображения ошибок.
- **Select**: Стилизованный выпадающий список.
- **Checkbox & Radio**: Кастомные элементы выбора.
- **Switch**: Переключатель для логических настроек.
- **DatePicker**: Поле выбора даты с поддержкой темной темы.
- **Textarea**: Поле для многострочного ввода.

## Управление состоянием
- **Авторизация**: Состояние `isLoggedIn` в `App.tsx` определяет доступ к основному контенту.
- **Данные пользователя**: Объект `UserData` хранит информацию, введенную на странице входа (ФИО, Email, Дата рождения, Пол).
- **Навигация**: Локальное состояние `showUIKit` переключает отображение между профилем пользователя и библиотекой компонентов.

## Анимации
- Используется `AnimatePresence` для плавного переключения между `LoginPage`, `Hero` (профиль) и `UIKit`.
- Применяются переходы `opacity`, `y` (смещение по вертикали) и `x` (смещение по горизонтали) для создания ощущения глубины и динамики.
